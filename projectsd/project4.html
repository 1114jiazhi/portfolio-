Event Explorer App - Real-Time Event Insights

This project demonstrates my ability to integrate multiple real-world data sources through APIs and web scraping, creating an interactive app that enriches users' event experiences by offering comprehensive, real-time insights.

Project Overview

The Event Explorer app pulls real-time data from Ticketmaster's API to display upcoming concerts and events based on user location. It supplements this information with:

  • Venue background and history (scraped from Wikipedia)
  • Nearby restaurant recommendations (via Google Places API)
  • Recent news about artists (scraped from entertainment news websites)
Technologies and Skills Demonstrated
  • API Authentication and Key Management (Ticketmaster API, Google API)
  • Data Caching to minimize redundant API calls and improve performance
  • HTTP Request Handling with `requests` and JSON Parsing
  • Web Scraping using `BeautifulSoup` for dynamic Wikipedia and news content extraction
  • Data Integration across multiple APIs and web-scraped sources
  • Error Handling, Rate Limiting, and Data Cleaning
Reflection

This project strengthened my ability to work with unstructured data sources, handle real-world API constraints, and design a functional product from scratch. It emphasized the importance of clean data pipelines, user-centric information delivery, and creative problem solving using multiple technologies.
